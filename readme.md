# React template

Prerequisites:

* Install dependencies:

  ```shell
  yarn install
  ```

## Start development server

```shell
yarn serve-dev
```

## Build for production

```shell
yarn build-prod
```

## Compile with TypeScript

Ensure that type system works (no files will be generated):

```shell
yarn type
```

## Lint with ESLint

```shell
yarn lint
```

## Format with Prettier

```shell
yarn format
```
